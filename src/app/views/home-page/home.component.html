Welcome to My ToDo App

This is a CRUD application built with Angular.
As required in this application you are going to find two routing functional pages.

  • Emplementing JWT as authentication for API calls.
  • Guards to access the different views requiring to validate the JWT against the API.
  • Reactive form validation for the form fields.

These pages are:

  • CRUD ACTIONS
  • GRID VIEW

can also access these pages via the top menu
